# Book-Stay-App

Book My Stay App is a Hotel Booking Management System built as an app-based learning project using Core Java and fundamental data structures.

## Objective

This project demonstrates how Core Java and data structures solve real-world software engineering problems such as:

- fair request handling using FIFO principles
- real-time room inventory consistency
- prevention of double-booking through uniqueness enforcement
- extensible and maintainable object-oriented design

## Learning Focus

- Core Java
- Object-oriented design
- Data structures in practical business workflows
- Deterministic console-based execution

## Planned Branch Strategy

- `main` for stable milestones
- `develop` for integrated work
- `feature/core-model`
- `feature/booking-service`
- `feature/data-structures`

## Initial Modules

- room management
- guest management
- booking workflow
- waitlist handling
- reporting

## Use Case 1

The first use case establishes the application entry point for `Book My Stay App`.
It demonstrates:

- Java class structure
- `main()` as the JVM entry point
- `static` execution without object creation
- console output with `System.out.println()`
- JavaDoc-based class documentation

Current behavior:

- prints a welcome message
- displays the application name
- displays version information
- terminates cleanly after execution
